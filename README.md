# Life3D

A 3D Game of Life for Apple Vision Pro — a mesmerizing spatial art piece built with cellular automata.

Luminous cells float in volumetric space, evolving through emergent complexity. Interact with your hands, choose patterns, and watch life unfold around you.

## Requirements

- Apple Vision Pro
- visionOS 2.0+
- Xcode 16+

## Building

```bash
open Life3D.xcodeproj
# Build for visionOS Simulator or device
```
